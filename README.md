# Course Platform

Educational content delivery platform for technical courses.

## Overview

Course Platform provides infrastructure for creating and delivering technical courses with:

- Video content hosting and streaming
- Interactive code exercises
- Progress tracking
- Certificate generation
- Community features

## Quick Start

```bash
uv sync
uv run course-platform serve
```

## Tech Stack

- Backend: FastAPI + PostgreSQL
- Frontend: React + Vite
- Video: Cloudflare Stream
- Auth: Clerk/Auth0
